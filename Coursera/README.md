#### ¿Qué es una clase?
	Es una abstracción del mundo real.
	Su declaración es:
	```
	public class Pizza
	{
		//Contenido de la clase
	}
	```
#### ¿Qué es un atributo?	
	Nos permiten definir las características o propiedades de una clase. 
	Los atributos
